# ACP CLIs Intent

## Purpose

Choose compatible ACP CLI surfaces for the selected harness.

## Required visible language

- Copilot
- ACP CLI

## Inputs and defaults

- CLI selection

## Required actions

- Select Copilot for the current Hermes lane
- Continue to providers after a compatible CLI is selected

## Blocking behavior

- At least one compatible CLI must be selected before continuing.

## Setup payload expectations

- Selected CLIs are included in the setup payload.

## Visual expectations

- Available and selected CLIs are visually clear.
