# Providers Intent

## Purpose

Choose model/provider integrations compatible with the selected agent surfaces in the Dynamic workflows lane.

## Required visible language

- Anthropic
- provider

## Inputs and defaults

- Provider selection

## Required actions

- Select Anthropic or another compatible provider
- Continue to model selection or provider configuration

## Blocking behavior

- At least one compatible provider must be selected before continuing.

## Setup payload expectations

- Selected providers are included in the setup payload.

## Visual expectations

- The selected provider is visible and distinguishable.
