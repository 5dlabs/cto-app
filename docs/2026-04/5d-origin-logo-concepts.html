5D Origin logo directions

Universe-native source control: Git origin meets cosmic origin, branches, gravity, and a black-hole event horizon. Palette harmonizes with the current neon cyan/violet 5D Labs mark.

Recommended primary mark

Event Horizon Branch

A dark origin core with cyan/violet source-control branches bending through an accretion disk.

  • Strongest black-hole + Git-origin meaning.
  • Memorable app-card silhouette.
  • Feels native to 5D’s neon portal language.
Technical alternate

Singularity Commit Graph

A commit graph spirals into the first source point: every branch traces back to origin.

  • Most source-control-specific.
  • Great for animation/loading states.
  • Risks becoming busy at very small sizes.
Small icon system glyph

Gravity Fork Glyph

A source-control fork warped around a black-hole core. Simpler, utility-first, high legibility.

  • Best toolbar/sidebar version.
  • Easy to reduce to one color.
  • Less cinematic than the primary mark.
Recommended direction: ship Event Horizon Branch as the primary 5D Origin logo, then simplify it into Gravity Fork Glyph for tiny UI sizes. Use Singularity Commit Graph only for motion/splash or marketing moments where detail can breathe.
