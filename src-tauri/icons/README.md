# CTO Lite Icons

Replace these placeholder files with actual icons:

- `icon.png` - 512x512 base icon
- `32x32.png` - 32x32 for small displays
- `128x128.png` - 128x128 for medium displays  
- `128x128@2x.png` - 256x256 for Retina displays
- `icon.icns` - macOS app icon bundle
- `icon.ico` - Windows icon

Generate from a source SVG using:
```bash
# macOS
iconutil -c icns icon.iconset

# Or use a tool like https://icon.kitchen
```
